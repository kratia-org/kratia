# ⚡ Kratia Frontend

**Ultra-fast, Resumable ERP Dashboard built with Qwik.**

---

## 🎨 Design Philosophy

- **Resumability**: Zero bundle size hydration for instant interactivity.
- **Modern Styling**: Powered by **Tailwind CSS v4** for a premium look and feel.
- **Form Handling**: Complex ERP forms managed by **Modular Forms** with SDK-level validation.

## 🛠 Tech Stack

- **Framework**: [Qwik City](https://qwik.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Forms**: [Modular Forms](https://modularforms.dev/)
- **Icons**: [Lucide React](https://lucide.dev/) (via Qwik wrapper)

## 🏗 Project Structure

- **`src/routes/`**: Directory-based routing and layouts.
- **`src/components/`**: Atomic design system components.
- **`src/core/`**: API clients and frontend-specific business logic.

## 🚀 Development

```bash
# Start development server
bun dev

# Preview production build
bun preview
```
